## Citation form

Within this chapter, decimal labels `(N.M.K)` cite the local volume. Cross-volume references appear verbatim as in EGA:
`(I, 4.2.3)` cites EGA I; `(0, 4.2.4)` cites Chapter 0 (preliminaries) of EGA I; `(III, 2.3.8)` cites EGA III. The
numbering matches the IHÉS print exactly, so a reader can follow a reference back into the French.
