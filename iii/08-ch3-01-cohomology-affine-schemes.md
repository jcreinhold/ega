# §1. Cohomology of affine schemes

## 1.1. Review of the exterior algebra complex

**1.1.1.**

<!-- label: III.1.1.1 -->

Let `A` be a ring, `𝐟 = (f_i)_{1 ≤ i ≤ r}` a system of `r` elements of `A`. The *exterior algebra complex* `K_•(𝐟)`
corresponding to `𝐟` is a chain complex `(G, I, 2.2)` defined as follows: the graded `A`-module `K_•(𝐟)` is equal to the
*exterior algebra* `Λ(A^r)`, graded in the usual way, and the boundary operator is the *interior multiplication* `i_𝐟`
by `𝐟`, viewed as an element of the dual `(A^r)^∨`. Recall that `i_𝐟` is an *antiderivation* of degree `−1` on `Λ(A^r)`,
and that if `(𝐞_i)_{1 ≤ i ≤ r}` is the canonical basis of `A^r` then `i_𝐟(𝐞_i) = f_i`; the verification of
`i_𝐟 ∘ i_𝐟 = 0` is immediate.

An equivalent definition is the following. For each `i`, consider the chain complex `K_•(f_i)` defined by
`K_0(f_i) = K_1(f_i) = A`, `K_n(f_i) = 0` for `n ≠ 0, 1`, and with boundary operator `d_1 : A → A` equal to
*multiplication by `f_i`*. Then `K_•(𝐟)` is the *tensor product*

```text
  K_•(𝐟) = K_•(f_1) ⊗ K_•(f_2) ⊗ ⋯ ⊗ K_•(f_r)                                (G, I, 2.7)
```

equipped with its total grading; the isomorphism between this complex and the one defined above is immediate.

> _Translator's note._ EGA's `complexe de l'algèbre extérieure` is what is now universally called the *Koszul complex*.
> We retain EGA's name on first introduction and use it interchangeably below.

**1.1.2.**

<!-- label: III.1.1.2 -->

For every `A`-module `M`, define the *chain complex*

```text
  K_•(𝐟, M) = K_•(𝐟) ⊗_A M                                                   (1.1.2.1)
```

and the *cochain complex* `(G, I, 2.2)`

```text
  K^•(𝐟, M) = Hom_A(K_•(𝐟), M).                                              (1.1.2.2)
```

If `g` is a `k`-cochain of the latter complex, and if we set

```text
  g(i_1, …, i_k) = g(𝐞_{i_1} ∧ ⋯ ∧ 𝐞_{i_k}),
```

then `g` identifies with an *alternating* map from `[1, r]^k` to `M`, and the above definitions give

```text
  d^k g(i_1, i_2, …, i_{k+1})
        = Σ_{h=1}^{k+1} (−1)^{h−1} f_{i_h} g(i_1, …, î_h, …, i_{k+1}).        (1.1.2.3)
```

<!-- original page 83 -->

**1.1.3.**

<!-- label: III.1.1.3 -->

From the preceding complexes one deduces, as usual, the *homology* and *cohomology* `A`-modules `(G, I, 2.2)`

```text
  H_•(𝐟, M) = H_•(K_•(𝐟, M)),                                                (1.1.3.1)
  H^•(𝐟, M) = H^•(K^•(𝐟, M)).                                                (1.1.3.2)
```

One defines an `A`-isomorphism `K_•(𝐟, M) ⥲ K^•(𝐟, M)` by associating to each chain
`z = Σ (𝐞_{i_1} ∧ ⋯ ∧ 𝐞_{i_k}) ⊗ z_{i_1, …, i_k}` the cochain `g_z` with

```text
  g_z(j_1, …, j_{r−k}) = ε · z_{i_1, …, i_k},
```

where `(j_h)_{1 ≤ h ≤ r−k}` is the strictly increasing sequence complementary to the strictly increasing sequence
`(i_h)_{1 ≤ h ≤ k}` in `[1, r]` and `ε = (−1)^ν`, with `ν` the number of inversions of the permutation
`i_1, …, i_k, j_1, …, j_{r−k}` of `[1, r]`. One checks that `g_{dz} = d(g_z)`, yielding an isomorphism

```text
  H^i(𝐟, M) ≅ H_{r−i}(𝐟, M)                  for 0 ≤ i ≤ r.                 (1.1.3.3)
```

In this chapter we shall mostly consider the cohomology modules `H^•(𝐟, M)`.

For a given `𝐟`, it is immediate `(G, I, 2.1)` that `M ↦ H^•(𝐟, M)` is a *cohomological functor* `(T, II, 2.1)` from the
category of `A`-modules to the category of graded `A`-modules, vanishing in degrees `< 0` and `> r`. In addition

```text
  H^0(𝐟, M) = Hom_A(A/(𝐟), M),                                               (1.1.3.4)
```

writing `(𝐟)` for the ideal of `A` generated by `f_1, …, f_r`; this follows immediately from `(1.1.2.3)`, and
`H^0(𝐟, M)` is clearly identified with the submodule of `M` annihilated by `(𝐟)`. Similarly, `(1.1.2.3)` gives

```text
  H^r(𝐟, M) = M / (Σ_{i=1}^r f_i M) = (A/(𝐟)) ⊗_A M.                         (1.1.3.5)
```

We shall use the following well-known result, whose proof we recall for completeness.

**Proposition (1.1.4).**

<!-- label: III.1.1.4 -->

*Let `A` be a ring, `𝐟 = (f_i)_{1 ≤ i ≤ r}` a finite family of elements of `A`, and `M` an `A`-module. If, for
`1 ≤ i ≤ r`, the multiplication `z ↦ f_i z` on `M_{i−1} = M / (f_1 M + ⋯ + f_{i−1} M)` is injective, then
`H^p(𝐟, M) = 0` for `p ≠ r`.*

**Proof.** It suffices to prove that `H_i(𝐟, M) = 0` for every `i > 0` by virtue of `(1.1.3.3)`. We argue by induction
on `r`, the case `r = 0` being trivial. Set `𝐟' = (f_i)_{1 ≤ i ≤ r−1}`; this family satisfies the hypotheses of the
statement, so on setting `L_• = K_•(𝐟', M)` we have `H_i(L_•) = 0` for `i > 0` by induction and `H_0(L_•) = M_{r−1}` by
`(1.1.3.3)` and `(1.1.3.5)`. To abbreviate, set `K_• = K_•(f_r) = K_0 ⊕ K_1`, with `K_0 = K_1 = A` and `d_1 : K_1 → K_0`
equal to multiplication by `f_r`; by definition (1.1.1) we have `K_•(𝐟, M) = K_• ⊗_A L_•`. We invoke the following
lemma.

**Lemma (1.1.4.1).**

<!-- label: III.1.1.4.1 -->

*Let `K_•` be a chain complex of free `A`-modules, zero except in dimensions `0` and `1`. For every chain complex `L_•`
of `A`-modules, there is an exact sequence*

```text
  0 → H_0(K_• ⊗ H_p(L_•)) → H_p(K_• ⊗ L_•) → H_1(K_• ⊗ H_{p−1}(L_•)) → 0
```

*for every index `p`.*

<!-- original page 84 -->

This is a particular case of an exact sequence of low-degree terms of the Künneth spectral sequence `(M, XVII, 5.2 (a))`
and `(G, I, 5.5.2)`; one can also prove it directly, as follows. Regard `K_0` and `K_1` as chain complexes (zero in
dimensions `≠ 0` and `≠ 1` respectively); then there is an exact sequence of complexes

```text
  0 → K_0 ⊗ L_• → K_• ⊗ L_• → K_1 ⊗ L_• → 0,
```

to which the exact sequence in homology applies:

```text
  ⋯ → H_{p+1}(K_1 ⊗ L_•) → H_p(K_0 ⊗ L_•) → H_p(K_• ⊗ L_•) → H_p(K_1 ⊗ L_•)
                       → H_{p−1}(K_0 ⊗ L_•) → ⋯.
```

But `H_p(K_0 ⊗ L_•) = K_0 ⊗ H_p(L_•)` and `H_p(K_1 ⊗ L_•) = K_1 ⊗ H_{p−1}(L_•)` for every `p`; one verifies immediately,
in addition, that the boundary `∂ : K_1 ⊗ H_p(L_•) → K_0 ⊗ H_p(L_•)` coincides with `d_1 ⊗ 1`. The lemma follows from
the exact sequence above and from the definitions of `H_0(K_• ⊗ H_p(L_•))` and `H_1(K_• ⊗ H_{p−1}(L_•))`.

With this lemma in hand, the rest of the proof of (1.1.4) is immediate: the induction hypothesis combined with (1.1.4.1)
gives `H_p(K_• ⊗ L_•) = 0` for `p ≥ 2`; moreover, if we show that `H_1(K_•, H_0(L_•)) = 0`, then the same lemma also
gives `H_1(K_• ⊗ L_•) = 0`. But by definition `H_1(K_•, H_0(L_•))` is the kernel of the multiplication `z ↦ f_r z` on
`M_{r−1}`, and that kernel is zero by hypothesis. The proof is complete.

**1.1.5.**

<!-- label: III.1.1.5 -->

Let `𝐠 = (g_i)_{1 ≤ i ≤ r}` be a second sequence of `r` elements of `A`, and set `𝐟𝐠 = (f_i g_i)_{1 ≤ i ≤ r}`. One can
define a canonical homomorphism of complexes

```text
  φ_𝐠 : K_•(𝐟𝐠) → K_•(𝐟)                                                     (1.1.5.1)
```

as the canonical extension to the exterior algebra `Λ(A^r)` of the `A`-linear map
`(x_1, …, x_r) ↦ (g_1 x_1, …, g_r x_r)` from `A^r` to itself. To see that this is a homomorphism of complexes, note in
general that if `u : E → F` is an `A`-linear map, `𝐱 ∈ F^∨`, and `𝐲 = ᵗu(𝐱) ∈ E^∨`, then

```text
  (Λ u) ∘ i_𝐲 = i_𝐱 ∘ (Λ u);                                                 (1.1.5.2)
```

indeed, both sides are antiderivations of `Λ F`, and it suffices to check that they coincide on `F`, which follows
immediately from the definitions.

When one identifies `K_•(𝐟)` with the tensor product of the `K_•(f_i)` (1.1.1), `φ_𝐠` is the tensor product of the
`φ_{g_i}`, where `φ_{g_i}` reduces to the identity in degree `0` and to multiplication by `g_i` in degree `1`.

**1.1.6.**

<!-- label: III.1.1.6 -->

In particular, for every pair of integers `m`, `n` with `0 ≤ n ≤ m`, we have homomorphisms of complexes

```text
  φ_{𝐟^{m−n}} : K_•(𝐟^m) → K_•(𝐟^n)                                          (1.1.6.1)
```

and hence homomorphisms

```text
  φ_{𝐟^{m−n}} : K^•(𝐟^n, M) → K^•(𝐟^m, M),                                   (1.1.6.2)
  φ_{𝐟^{m−n}} : H^•(𝐟^n, M) → H^•(𝐟^m, M).                                   (1.1.6.3)
```

<!-- original page 85 -->

The latter satisfy the obvious transitivity relation `φ_{𝐟^{m−p}} = φ_{𝐟^{m−n}} ∘ φ_{𝐟^{n−p}}` for `p ≤ n ≤ m`; they
thus define two inductive systems of `A`-modules. We set

```text
  C^•((𝐟), M) = lim_→ K^•(𝐟^n, M),                                           (1.1.6.4)
                  n
  H^•((𝐟), M) = H^•(C^•((𝐟), M)) = lim_→ H^•(𝐟^n, M),                        (1.1.6.5)
                                     n
```

the last equality coming from the fact that the inductive limit commutes with the functor `H^•` `(G, I, 2.1)`. We shall
see later (1.4.3) that `H^•((𝐟), M)` depends only on the *ideal* `(𝐟)` of `A` (in fact only on the `(𝐟)`-preadic
topology on `A`), which justifies the notations.

It is clear that `M ↦ C^•((𝐟), M)` is an exact `A`-linear functor and `M ↦ H^•((𝐟), M)` a cohomological functor.

**1.1.7.**

<!-- label: III.1.1.7 -->

Let `𝐟 = (f_i) ∈ A^r` and `𝐠 = (g_i) ∈ A^r`; write `e_𝐠` for the left multiplication by the vector `𝐠 ∈ A^r` on the
exterior algebra `Λ(A^r)`. We have the *homotopy formula*

```text
  i_𝐟 e_𝐠 + e_𝐠 i_𝐟 = ⟨𝐠, 𝐟⟩ · 1                                            (1.1.7.1)
```

on the `A`-module `Λ(A^r)` (where `1` denotes the identity automorphism of `Λ(A^r)`); this relation says, equivalently,
that *in the complex `K_•(𝐟)`*

```text
  d e_𝐠 + e_𝐠 d = ⟨𝐠, 𝐟⟩ · 1.                                                (1.1.7.2)
```

If the ideal `(𝐟)` is equal to `A`, then there exists `𝐠 ∈ A^r` such that `⟨𝐠, 𝐟⟩ = Σ_{i=1}^r g_i f_i = 1`. Therefore
`(G, I, 2.4)`:

**Proposition (1.1.8).**

<!-- label: III.1.1.8 -->

*Suppose the ideal `(𝐟)` generated by the `f_i` is equal to `A`. Then the complex `K_•(𝐟)` is homotopically trivial, and
so are the complexes `K_•(𝐟, M)` and `K^•(𝐟, M)` for every `A`-module `M`.*

**Corollary (1.1.9).**

<!-- label: III.1.1.9 -->

*If `(𝐟) = A`, then `H^•(𝐟, M) = 0` and `H^•((𝐟), M) = 0` for every `A`-module `M`.*

**Proof.** We then have `(𝐟^n) = A` for every `n`.

**Remark (1.1.10).**

<!-- label: III.1.1.10 -->

With the notation above, let `X = Spec(A)`, and let `Y` be the closed subscheme of `X` defined by the ideal `(𝐟)`. We
shall prove in §9 that `H^•((𝐟), M)` is isomorphic to the cohomology `H_Y^•(X, M̃)` of `X` with support in the filter of
closed subsets of `Y` `(T, 3.2)`. We shall also show that Proposition (1.2.3), applied to `X` and to `ℱ = M̃`, is a
particular case of an exact cohomology sequence

```text
  ⋯ → H_Y^p(X, ℱ) → H^p(X, ℱ) → H^p(X − Y, ℱ) → H_Y^{p+1}(X, ℱ) → ⋯.
```

## 1.2. Čech cohomology of an open cover

**1.2.1. Notations.**

<!-- label: III.1.2.1 -->

Throughout this article we shall write:

- `X` for a prescheme;
- `ℱ` for a quasi-coherent `𝒪_X`-module;

<!-- original page 86 -->

- `A = Γ(X, 𝒪_X)`, `M = Γ(X, ℱ)`;
- `𝐟 = (f_i)_{1 ≤ i ≤ r}` for a finite system of elements of `A`;
- `U_i = X_{f_i}` for the open set `(0_I, 5.5.2)` of `x ∈ X` with `f_i(x) ≠ 0`;
- `U = ⋃ U_i`;
- `𝔘` for the open cover `(U_i)_{1 ≤ i ≤ r}` of `U`.

**1.2.2.**

<!-- label: III.1.2.2 -->

Suppose `X` is either a prescheme whose underlying space is Noetherian, or a scheme whose underlying space is
quasi-compact. Then `(I, 9.3.3)` gives `Γ(U_i, ℱ) = M_{f_i}`. Set

```text
  U_{i_0, …, i_p} = ⋂_{k=0}^p U_{i_k} = X_{f_{i_0} ⋯ f_{i_p}}                (0_I, 5.5.3);
```

we then likewise have

```text
  Γ(U_{i_0, …, i_p}, ℱ) = M_{f_{i_0} ⋯ f_{i_p}}.                             (1.2.2.1)
```

But `(0_I, 1.6.1)` identifies `M_{f_{i_0} ⋯ f_{i_p}}` with the inductive limit `lim_→_n M_{i_0, …, i_p}^{(n)}`, where
`M_{i_0, …, i_p}^{(n)} = M` and the transition map `φ_{n,m} : M_{i_0, …, i_p}^{(m)} → M_{i_0, …, i_p}^{(n)}` is
multiplication by `(f_{i_0} ⋯ f_{i_p})^{n−m}` for `m ≤ n`. Denote by `C^p_n(M)` the set of alternating maps from
`[1, r]^{p+1}` to `M` (for every `n`); these `A`-modules likewise form an inductive system under the `φ_{n,m}`. If
`C^p(𝔘, ℱ)` is the group of alternating Čech `p`-cochains of `ℱ` relative to the cover `𝔘` `(G, II, 5.1)`, the foregoing
gives

```text
  C^p(𝔘, ℱ) = lim_→ C^p_n(M).                                                 (1.2.2.2)
                n
```

With the notation of (1.1.2), `C^p_n(M)` is identified with `K^{p+1}(𝐟^n, M)`, and the map `φ_{n,m}` with the map
`φ_{𝐟^{n−m}}` defined in (1.1.6). Thus, for every `p ≥ 0`, there is a canonical functorial-in-`ℱ` isomorphism

```text
  C^p(𝔘, ℱ) ≅ C^p((𝐟), M).                                                   (1.2.2.3)
```

Moreover, the formula `(1.1.2.3)` and the definition of the cohomology of a cover `(G, II, 5.1)` show that the
isomorphisms `(1.2.2.3)` are compatible with the coboundary operators.

**Proposition (1.2.3).**

<!-- label: III.1.2.3 -->

*If `X` is a prescheme whose underlying space is Noetherian, or a scheme whose underlying space is quasi-compact, then
there is a canonical functorial-in-`ℱ` isomorphism*

```text
  H^p(𝔘, ℱ) ≅ H^{p+1}((𝐟), M)              for every p ≥ 1.                  (1.2.3.1)
```

*In addition, there is a functorial-in-`ℱ` exact sequence*

```text
  0 → H^0((𝐟), M) → M → H^0(𝔘, ℱ) → H^1((𝐟), M) → 0.                         (1.2.3.2)
```

<!-- original page 87 -->

**Proof.** The relations `(1.2.3.1)` are an immediate consequence of (1.2.2). On the other hand,
`C^0(𝔘, ℱ) = C^1((𝐟), M)`, so `H^0(𝔘, ℱ)` identifies with the subgroup of `1`-cocycles of `C^•((𝐟), M)`; since
`M = C^0((𝐟), M)`, the exact sequence `(1.2.3.2)` is the one that comes from the definition of the cohomology groups
`H^0((𝐟), M)` and `H^1((𝐟), M)`.

**Corollary (1.2.4).**

<!-- label: III.1.2.4 -->

*Suppose the `X_{f_i}` are quasi-compact and that there exist `g_i ∈ Γ(U, ℱ)` such that `Σ_i g_i (f_i ∣ U) = 1_U`. Then,
for every quasi-coherent `(𝒪_X ∣ U)`-module `ℱ`, we have `H^p(𝔘, ℱ) = 0` for `p > 0`; if in addition `U = X`, the
canonical homomorphism `(1.2.3.2)` `M → H^0(𝔘, ℱ)` is bijective.*

**Proof.** Since by hypothesis the `U_i = X_{f_i}` are quasi-compact, so is `U`, so we may reduce to the case `U = X`;
the hypothesis then implies `H^p((𝐟), M) = 0` for every `p ≥ 0` (1.1.9). The corollary follows from `(1.2.3.1)` and
`(1.2.3.2)`.

Note that since `H^0(𝔘, ℱ) = H^0(U, ℱ)` `(G, II, 5.2.2)`, the result `(I, 1.3.7)` is recovered here as a special case.

**Remark (1.2.5).**

<!-- label: III.1.2.5 -->

Suppose `X` is an affine scheme; then the `U_i = X_{f_i} = D(f_i)` are affine open sets, as are the `U_{i_0, …, i_p}`
(though `U` is not necessarily affine). In this case the functors `Γ(X, ·)` and `Γ(U_{i_0, …, i_p}, ·)` are exact in `ℱ`
`(I, 1.3.11)`. Given an exact sequence `0 → ℱ' → ℱ → ℱ'' → 0` of quasi-coherent `𝒪_X`-modules, the sequence of complexes

```text
  0 → C^•(𝔘, ℱ') → C^•(𝔘, ℱ) → C^•(𝔘, ℱ'') → 0
```

is exact, and yields a long exact sequence in cohomology

```text
  ⋯ → H^p(𝔘, ℱ') → H^p(𝔘, ℱ) → H^p(𝔘, ℱ'') → H^{p+1}(𝔘, ℱ') → ⋯.
```

On the other hand, setting `M' = Γ(X, ℱ')`, `M'' = Γ(X, ℱ'')`, the sequence `0 → M' → M → M'' → 0` is exact; as
`C^•((𝐟), M)` is an exact functor of `M`, we also obtain the exact sequence in cohomology

```text
  ⋯ → H^p((𝐟), M') → H^p((𝐟), M) → H^p((𝐟), M'') → H^{p+1}((𝐟), M') → ⋯.
```

Since the diagram

```text
  0 → C^•(𝔘, ℱ')     → C^•(𝔘, ℱ)     → C^•(𝔘, ℱ'')     → 0
        ↓                 ↓                 ↓
  0 → C^•((𝐟), M')   → C^•((𝐟), M)   → C^•((𝐟), M'')   →
```

<!-- original page 88 -->

is commutative, the diagrams

```text
              H^p(𝔘, ℱ'')   →   H^{p+1}(𝔘, ℱ')
                  ↓                  ↓                                       (1.2.5.1)
              H^{p+1}((𝐟), M'') → H^{p+1}((𝐟), M')
```

are commutative for every `p` `(G, I, 2.1.1)`.

## 1.3. Cohomology of an affine scheme

**Theorem (1.3.1).**

<!-- label: III.1.3.1 -->

*Let `X` be an affine scheme. For every quasi-coherent `𝒪_X`-module `ℱ`, we have `H^p(X, ℱ) = 0` for every `p > 0`.*

**Proof.** Let `𝔘` be a finite cover of `X` by affine open sets `X_{f_i} = D(f_i)` (`1 ≤ i ≤ r`); the ideal of
`A = Γ(X, 𝒪_X)` generated by the `f_i` is then `A`. By `(1.2.4)` we have `Ȟ^p(𝔘, ℱ) = 0` for `p > 0`. Since there are
arbitrarily fine finite affine open covers of `X` `(I, 1.1.10)`, the definition of Čech cohomology `(G, II, 5.8)` shows
that `Ȟ^p(X, ℱ) = 0` for `p > 0`. The same argument applies to every `X_f` for `f ∈ A` `(I, 1.3.6)`, so
`Ȟ^p(X_f, ℱ) = 0` for `p > 0`. As `X_f ∩ X_g = X_{fg}`, we conclude that `H^p(X, ℱ) = 0` for every `p > 0` by virtue of
`(G, II, 5.9.2)`.

**Corollary (1.3.2).**

<!-- label: III.1.3.2 -->

*Let `Y` be a prescheme and `f : X → Y` an affine morphism `(II, 1.6.1)`. For every quasi-coherent `𝒪_X`-module `ℱ`, we
have `R^q f_*(ℱ) = 0` for `q > 0`.*

**Proof.** By definition, `R^q f_*(ℱ)` is the `𝒪_Y`-module associated to the presheaf `U ↦ H^q(f^{-1}(U), ℱ)`, where `U`
ranges over the open sets of `Y`. The affine open sets form a base of `Y`, and for such a `U` the preimage `f^{-1}(U)`
is affine `(II, 1.3.2)`, hence `H^q(f^{-1}(U), ℱ) = 0` by `(1.3.1)`, proving the corollary.

**Corollary (1.3.3).**

<!-- label: III.1.3.3 -->

*Let `Y` be a prescheme and `f : X → Y` an affine morphism. For every quasi-coherent `𝒪_X`-module `ℱ`, the canonical
homomorphism*

```text
  H^p(Y, f_*(ℱ)) → H^p(X, ℱ)                                   (0, 12.1.3.1)
```

*is bijective for every `p`.*

**Proof.** By `(0, 12.1.7)` it suffices to show that the edge homomorphisms `″E_2^{p, 0} = H^p(Y, f_*(ℱ)) → H^p(X, ℱ)`
of the second spectral sequence of the composite functor `Γ f_*` are bijective. The `E_2` term of this sequence is
`″E_2^{p, q} = H^p(Y, R^q f_*(ℱ))` `(G, II, 4.17.1)`, and (1.3.2) gives `″E_2^{p, q} = 0` for `q > 0`; the spectral
sequence therefore degenerates. The assertion follows from `(0, 11.1.6)`.

**Corollary (1.3.4).**

<!-- label: III.1.3.4 -->

*Let `f : X → Y` be an affine morphism and `g : Y → Z` a morphism.*

<!-- original page 89 -->

*For every quasi-coherent `𝒪_X`-module `ℱ`, the canonical homomorphism `R^p g_*(f_*(ℱ)) → R^p(g ∘ f)_*(ℱ)`
`(0, 12.2.5.1)` is bijective for every `p`.*

**Proof.** Note that, by (1.3.3), for every affine open `W` of `Z`, the canonical homomorphism
`H^p(g^{-1}(W), f_*(ℱ)) → H^p(f^{-1}(g^{-1}(W)), ℱ)` is bijective; this shows that the presheaf homomorphism defining
the canonical homomorphism `R^p g_*(f_*(ℱ)) → R^p(g ∘ f)_*(ℱ)` is bijective `(0, 12.2.5)`.

## 1.4. Application to the cohomology of arbitrary preschemes

**Proposition (1.4.1).**

<!-- label: III.1.4.1 -->

*Let `X` be a scheme and `𝔘 = (U_i)` a cover of `X` by affine open sets. For every quasi-coherent `𝒪_X`-module `ℱ`, the
cohomology modules `H^•(X, ℱ)` and `H^•(𝔘, ℱ)` (over `Γ(X, 𝒪_X)`) are canonically isomorphic.*

**Proof.** Since `X` is a scheme, every finite intersection `V` of open sets of the cover `𝔘` is affine `(I, 5.5.6)`, so
`H^q(V, ℱ) = 0` for `q ≥ 1` by `(1.3.1)`. The proposition follows from Leray's theorem `(G, II, 5.4.1)`.

**Remark (1.4.2).**

assumed to be a scheme.

**Corollary (1.4.3).**

<!-- label: III.1.4.3 -->

*and a functorial-in-`ℱ` exact sequence*


**Proof.** This follows from (1.4.1) and (1.2.3).

**Proposition (1.4.5).**

<!-- label: III.1.4.5 -->

*Let `X` be a quasi-compact scheme, `ℒ` an invertible `𝒪_X`-module, and consider the graded ring `A_• = Γ_*(ℒ)`
`(II, 5.4.6)`. Then `H^•(X, ℱ) = ⊕_n H^•(X, ℱ ⊗ ℒ^{⊗n})` is a graded `A_•`-module, and for every `f ∈ A_d` there is a
canonical isomorphism*

```text
  H^•(X_f, ℱ) ≅ (H^•(X, ℱ))_{(f)}                                            (1.4.5.1)
```

*of `(A_•)_{(f)}`-modules.*

**Proof.** Since `X` is a quasi-compact scheme, the cohomology of all the `𝒪_X`-modules `ℱ ⊗ ℒ^{⊗n}` can be computed
with one and the same finite cover `𝔘 = (U_i)` by affine open sets such that the restriction `ℒ ∣ U_i` is isomorphic to
`𝒪_X ∣ U_i` for each `i` (1.4.1). It is then immediate that the `U_i ∩ X_f` are affine open sets `(I, 1.3.6)`, so the
cohomology `H^•(X_f, ℱ ⊗ ℒ^{⊗n})` can also be computed with the cover `𝔘 ∣ X_f = (U_i ∩ X_f)` (1.4.1). It is immediate
that for every `f ∈ A_d` multiplication by `f` defines a homomorphism `C^•(𝔘, ℱ ⊗ ℒ^{⊗n}) → C^•(𝔘, ℱ ⊗ ℒ^{⊗(n+d)})`,
hence a homomorphism `H^•(𝔘, ℱ ⊗ ℒ^{⊗n}) → H^•(𝔘, ℱ ⊗ ℒ^{⊗(n+d)})`, which establishes the first assertion. On the other
hand, for `f ∈ A_d`, `(I, 9.3.2)` gives an isomorphism of complexes of `(A_•)_{(f)}`-modules

```text
  C^•(𝔘 ∣ X_f, ℱ) ≅ (C^•(𝔘, ℱ ⊗ ℒ^{⊗•}))_{(f)},
```

taking `(I, 1.3.9, (ii))` into account. Passing to the cohomology of both complexes, one obtains the isomorphism
`(1.4.5.1)`, using that the functor `M ↦ M_{(f)}` is exact on the category of graded `A_•`-modules.

**Corollary (1.4.7).**

<!-- label: III.1.4.7 -->

*Let `X` be a quasi-compact scheme and `f ∈ Γ(X, 𝒪_X)`.*

*(i) Suppose the open set `X_f` is affine. Then, for every quasi-coherent `𝒪_X`-module `ℱ`, every `i > 0`, and every
`ξ ∈ H^i(X, ℱ)`, there exists an integer `n > 0` such that `f^n ξ = 0`.*

*(ii) Conversely, suppose `X_f` is quasi-compact and that for every quasi-coherent sheaf of ideals `𝒥` of `𝒪_X` and
every `ξ ∈ H^1(X, 𝒥)` there exists `n > 0` such that `f^n ξ = 0`. Then `X_f` is affine.*

**Proof.** (i) If `X_f` is affine, then `H^i(X_f, ℱ) = 0` for every `i > 0` (1.3.1), so the assertion follows from
(1.4.6).

(ii) By Serre's criterion `(II, 5.2.1)`, it suffices to show that for every quasi-coherent sheaf of ideals `𝒥'` of
`𝒪_{X_f}` we have `H^1(X_f, 𝒥') = 0`. Since `X_f` is a quasi-compact open set in the quasi-compact scheme `X`, there is
a quasi-coherent sheaf of ideals `𝒥` of `𝒪_X` such that `𝒥' = 𝒥 ∣ X_f` `(I, 9.4.2)`. By (1.4.6),
`H^1(X_f, 𝒥') = (H^1(X, 𝒥))_f`, and the hypothesis forces the right side to be zero, whence the conclusion.

**Lemma (1.4.9).**

<!-- label: III.1.4.9 -->

*Let `X` be a quasi-compact scheme, `𝔘 = (U_i)_{1 ≤ i ≤ n}` a finite cover of `X` by affine open sets, and `ℱ` a
quasi-coherent `𝒪_X`-module. The complex of sheaves `𝒞^•(𝔘, ℱ)` defined by the cover `𝔘` `(G, II, 5.2)` is then a
quasi-coherent `𝒪_X`-module.*

**Proof.** By the definitions `(G, II, 5.2)`, `𝒞^•(𝔘, ℱ)` is a direct sum of sheaves obtained as direct images of the
`ℱ ∣ U_{i_0, …, i_p}` under the canonical injections `U_{i_0, …, i_p} → X`. The hypothesis that `X` is a scheme implies
that these injections are affine morphisms `(I, 5.5.6)`, so the `𝒞^•(𝔘, ℱ)` are quasi-coherent `(II, 1.2.6)`.

**Proposition (1.4.10).**

<!-- label: III.1.4.10 -->

*Let `u : X → Y` be a separated and quasi-compact morphism. For every quasi-coherent `𝒪_X`-module `ℱ`, the `R^q u_*(ℱ)`
are quasi-coherent `𝒪_Y`-modules.*

**Proof.** The question being local on `Y`, we may suppose `Y` affine. Then `X` is a finite union of affine open sets
`U_i` (`1 ≤ i ≤ n`); let `𝔘 = (U_i)`. Since `Y` is a scheme, `(I, 5.5.10)` shows that for every affine open `V ⊆ Y` the
canonical injection `u^{-1}(V) → X` is an affine morphism; therefore (by (1.4.1) and `(G, II, 5.2)`) there is a
canonical isomorphism

```text
  H^•(u^{-1}(V), ℱ) ≅ H^•(Γ(V, 𝒢)),                                          (1.4.10.1)
```

where `𝒢 = u_*(𝒞^•(𝔘, ℱ))`. By (1.4.9) and `(I, 9.2.2)`, `𝒢` is a quasi-coherent `𝒪_Y`-module; moreover, since
`𝒞^•(𝔘, ℱ)` is a complex of sheaves, so is `𝒢`. The definition of the cohomology `𝓗^•(𝒢)` `(G, II, 4.1)` then shows that
the latter consists of quasi-coherent `𝒪_Y`-modules `(I, 4.1.1)`. Since, for `V` affine in `Y`, the functor `Γ(V, ·)` is
exact in `𝒢` on the category of quasi-coherent `𝒪_Y`-modules, `(G, II, 4.1)` gives

```text
  H^•(Γ(V, 𝒢)) = Γ(V, 𝓗^•(𝒢)).                                               (1.4.10.2)
```

Finally, the definition of the canonical homomorphism `H^•(𝔘, ℱ) → H^•(X, ℱ)` given in `(G, II, 5.2)` shows that, if
`V' ⊆ V` is a second affine open set in `Y`, the diagram

```text
  H^•(u^{-1}(V), ℱ)   ≅   H^•(Γ(V, 𝒢))
        ↓                       ↓
  H^•(u^{-1}(V'), ℱ)  ≅   H^•(Γ(V', 𝒢))
```

is commutative. We conclude from the foregoing that the isomorphisms `(1.4.10.1)` define an isomorphism of `𝒪_Y`-modules

```text
  R^•u_*(ℱ) ≅ 𝓗^•(𝒢),                                                        (1.4.10.3)
```

<!-- original page 92 -->

and hence `R^q u_*(ℱ)` is quasi-coherent. The relations `(1.4.10.3)`, `(1.4.10.2)`, `(1.4.10.1)` also yield:

**Corollary (1.4.11).**

<!-- label: III.1.4.11 -->

*Under the hypotheses of (1.4.10), for every affine open set `V` of `Y` the canonical homomorphism*

```text
  H^q(u^{-1}(V), ℱ) → Γ(V, R^q u_*(ℱ))                                       (1.4.11.1)
```

*is an isomorphism for every `q ≥ 0`.*

**Corollary (1.4.12).**

<!-- label: III.1.4.12 -->

*Under the hypotheses of (1.4.10), suppose in addition that `Y` is quasi-compact. Then there exists an integer `r > 0`
such that for every quasi-coherent `𝒪_X`-module `ℱ` and every integer `q > r` we have `R^q u_*(ℱ) = 0`. If `Y` is
affine, one may take for `r` any integer such that `X` admits an affine open cover of cardinality `r`.*

**Proof.** Since `Y` can be covered by finitely many affine open sets, it suffices, by (1.4.11), to prove the second
assertion. If `𝔘` is a cover of `X` by `r` affine open sets, then `C^q(𝔘, ℱ) = 0` for `q > r`, since the Čech cochains
in `C^q(𝔘, ℱ)` are alternating; the conclusion follows from (1.4.1).

**Corollary (1.4.13).**

<!-- label: III.1.4.13 -->

*Under the hypotheses of (1.4.10), suppose in addition that `Y = Spec(A)` is affine. Then, for every quasi-coherent
`𝒪_X`-module `ℱ` and every `f ∈ A`,*

```text
  Γ(Y_f, R^q u_*(ℱ)) = (Γ(Y, R^q u_*(ℱ)))_f
```

*up to canonical isomorphism.*

**Proof.** This follows from the fact that `R^q u_*(ℱ)` is a quasi-coherent `𝒪_Y`-module `(I, 1.3.7)`.

**Proposition (1.4.14).**

<!-- label: III.1.4.14 -->

*Let `f : X → Y` be a separated quasi-compact morphism, and `g : Y → Z` an affine morphism. For every quasi-coherent
`𝒪_X`-module `ℱ`, the canonical homomorphism*

```text
  R^p(g ∘ f)_*(ℱ) → g_*(R^p f_*(ℱ))                                  (0, 12.2.5.2)
```

*is bijective for every `p`.*

**Proof.** For every affine open set `W` of `Z`, `g^{-1}(W)` is an affine open set of `Y`. The presheaf homomorphism
defining the canonical homomorphism `R^p(g ∘ f)_*(ℱ) → g_*(R^p f_*(ℱ))` `(0, 12.2.5)` is therefore bijective by
(1.4.11).

**Proposition (1.4.15).**

<!-- label: III.1.4.15 -->

*Let `u : X → Y` be a separated morphism of finite type, `v : Y' → Y` a flat morphism of preschemes `(0_I, 6.7.1)`. Set
`u' = u_{(Y')}`, so that one has the commutative diagram*

```text
       X  ←  X' = X ×_Y Y'
       ↓        ↓                                                            (1.4.15.1)
       Y  ←  Y'
```

*Then, for every quasi-coherent `𝒪_X`-module `ℱ` and every `q ≥ 0`, `R^q u'_*(ℱ')`, where
`ℱ' = v'^*(ℱ) = ℱ ⊗_{𝒪_X} 𝒪_{X'}`, is canonically isomorphic to `R^q u_*(ℱ) ⊗_{𝒪_Y} 𝒪_{Y'} = v^*(R^q u_*(ℱ))`.*

<!-- original page 93 -->

**Proof.** The canonical homomorphism `ρ : ℱ → v'_*(v'^*(ℱ))` `(0_I, 4.4.3.2)` defines by functoriality a homomorphism

```text
  R^q u_*(ℱ) → R^q u_*(v'_*(ℱ')).                                            (1.4.15.2)
```

On the other hand, setting `w = u ∘ v' = v ∘ u'`, one has the canonical homomorphisms `(0, 12.2.5.1)` and
`(0, 12.2.5.2)`

```text
  R^q u_*(v'_*(ℱ')) → R^q w_*(ℱ') → v_*(R^q u'_*(ℱ')).                       (1.4.15.3)
```

Composing `(1.4.15.3)` and `(1.4.15.2)` gives a homomorphism

```text
  ψ : R^q u_*(ℱ) → v_*(R^q u'_*(ℱ'))
```

and hence the canonical homomorphism (whose definition uses no hypothesis on `v`)

```text
  Φ : v^*(R^q u_*(ℱ)) → R^q u'_*(ℱ'),                                        (1.4.15.4)
```

which one must show is an isomorphism when `v` is flat. The question is local on `Y` and `Y'`, so we may suppose
`Y = Spec(A)` and `Y' = Spec(B)`. We use the following lemma.

**Lemma (1.4.15.5).**

<!-- label: III.1.4.15.5 -->

*Let `φ : A → B` be a ring homomorphism, `Y = Spec(A)`, `X = Spec(B)`, `f : X → Y` the morphism corresponding to `φ`,
and `M` a `B`-module. The `𝒪_X`-module `M̃` is `f`-flat `(0_I, 6.7.1)` if and only if `M` is a flat `A`-module. In
particular, the morphism `f` is flat if and only if `B` is a flat `A`-module.*

**Proof.** This follows from the definition `(0_I, 6.7.1)` and from `(0_I, 6.3.3)`, taking `(I, 1.3.4)` into account.

That being so, by `(1.4.11.1)` and the definitions of the homomorphisms `(1.4.15.3)` (cf. `0, 12.2.5`), `Φ` corresponds
to the composite morphism

```text
  H^q(X, ℱ) →^ρ H^q(X, v'_*(ℱ')) →^σ H^q(X', v'^*(v'_*(ℱ'))) →^θ H^q(X', ℱ'),
```

where `ρ` and `σ` are the homomorphisms corresponding in cohomology to the canonical morphisms `ρ` and
`σ : v'^*(v'_*(ℱ')) → ℱ'`, and `θ` is the `φ`-morphism `(0, 12.1.3.1)` relative to the `𝒪_X`-module `v'_*(v'^*(ℱ))`. By
the functoriality of `θ`, one has the commutative diagram

```text
   H^q(X, ℱ)              →^{ρ_φ}              H^q(X, v'_*(v'^*(ℱ)))
        ↓                                          ↓
   H^q(X', v'^*(ℱ))                  H^q(X', v'^*(v'_*(v'^*(ℱ))))
```

<!-- original page 94 -->

and since by definition `(0_I, 4.4.3)` `v'^*(ρ)` is the inverse of `σ`, the composite morphism considered above is none
other than `θ_φ`; thus `Φ` is the `B`-homomorphism associated to `H^q(X, ℱ) ⊗_A B → H^q(X', ℱ')`. Since `u` is of finite
type, `X` is a finite union of affine open sets `U_i` (`1 ≤ i ≤ r`); let `𝔘 = (U_i)`. Moreover, `v` is an affine
morphism, hence so is `v'` `(II, 1.6.2, (iii))`, and the `U'_i = v'^{-1}(U_i)` form a finite affine open cover `𝔘'` of
`X'`. We know `(0, 12.1.4.2)` that the diagram

```text
   H^•(𝔘, ℱ)    →    H^•(𝔘', ℱ')
        ↓                  ↓
   H^•(X, ℱ)    →    H^•(X', ℱ')
```

is commutative, and the vertical arrows are isomorphisms, since `X` and `X'` are schemes (1.4.1). It thus suffices to
show that the canonical `φ`-morphism `θ : H^•(𝔘, ℱ) → H^•(𝔘', ℱ')` is such that the associated `B`-homomorphism
`H^•(𝔘, ℱ) ⊗_A B → H^•(𝔘', ℱ')` is an isomorphism. For every sequence `s = (i_h)_{0 ≤ h ≤ p}` of `p + 1` indices in
`[1, r]`, set `U_s = ⋂_h U_{i_h}`, `U'_s = ⋂_h U'_{i_h} = v'^{-1}(U_s)`, `M_s = Γ(U_s, ℱ)`, `M'_s = Γ(U'_s, ℱ')`. The
canonical map `M_s ⊗_A B → M'_s` is an isomorphism `(I, 1.6.5)`, so the canonical map `C^•(𝔘, ℱ) ⊗_A B → C^•(𝔘', ℱ')` is
an isomorphism, under which `d ⊗ 1` is identified with the coboundary operator `C^•(𝔘', ℱ') → C^{•+1}(𝔘', ℱ')`. Since
`B` is a flat `A`-module, the definition of the cohomology modules immediately gives that the canonical map
`H^•(𝔘, ℱ) ⊗_A B → H^•(𝔘', ℱ')` is an isomorphism `(0_I, 6.1.1)`. This result will be generalized later (§6).

**Corollary (1.4.16).**

<!-- label: III.1.4.16 -->

*Let `A` be a ring, `X` an `A`-scheme of finite type, `B` an `A`-algebra faithfully flat over `A`. Then `X` is affine if
and only if `X ⊗_A B` is affine.*

**Proof.** The condition is evidently necessary `(I, 3.2.2)`; we show it is sufficient. Since `X` is separated over `A`
and the morphism `Spec(B) → Spec(A)` is flat, (1.4.15) gives

```text
  H^i(X ⊗_A B, ℱ ⊗_A B) = H^i(X, ℱ) ⊗_A B                                    (1.4.16.1)
```

for every `i ≥ 0` and every quasi-coherent `𝒪_X`-module `ℱ`. If `X ⊗_A B` is affine, the left side of `(1.4.16.1)` is
zero for `i = 1`, hence so is `H^1(X, ℱ)`, since `B` is faithfully flat over `A`. As `X` is a quasi-compact scheme, we
conclude by Serre's criterion `(II, 5.2.1)`.

**Proposition (1.4.17).**

<!-- label: III.1.4.17 -->

*Let `X` be a prescheme and `0 → ℱ' → ℱ → ℱ'' → 0` an exact sequence of `𝒪_X`-modules. If `ℱ'` and `ℱ''` are
quasi-coherent, so is `ℱ`.*

<!-- original page 95 -->

**Proof.** The question is local on `X`, so we may suppose `X = Spec(A)` affine; it then suffices to show that `ℱ`
satisfies the conditions `(d_1)` and `(d_2)` of `(I, 1.4.1)` (with `V = X`). The verification of `(d_1)` is immediate:
if `t ∈ Γ(X, ℱ)` has zero restriction to `D(f)`, then so does its image `v(t) ∈ Γ(X, ℱ'')`; thus there exists `m > 0`
such that `f^m v(t) = v(f^m t) = 0` `(I, 1.4.1)`. Since `ℱ'` is exact at left, `f^m t = u(s)` for some `s ∈ Γ(X, ℱ')`;
since `u` is injective, the restriction of `s` to `D(f)` is zero, hence by `(I, 1.4.1)` there is `n > 0` with
`f^n s = 0`, and finally `f^{m+n} t = u(f^n s) = 0`.

Now verify `(d_2)`. Let `t' ∈ Γ(D(f), ℱ)`. Since `ℱ''` is quasi-coherent, there exists an integer `m` such that
`f^m v(t') = v(f^m t')` extends to a section `t'' ∈ Γ(X, ℱ'')` `(I, 1.4.1)`. By (1.3.1) (or `(I, 5.1.9.2)`) applied to
the quasi-coherent `𝒪_X`-module `ℱ''`, the sequence `Γ(X, ℱ) → Γ(X, ℱ'') → 0` is exact, so there exists `τ ∈ Γ(X, ℱ)`
with `v(τ) = t''`. We see that `v(f^m t' − τ ∣ D(f)) = 0`; writing `t'''` for the restriction of `τ` to `D(f)`, we have
`f^m t' − t''' = u(s')` for some `s' ∈ Γ(D(f), ℱ')`. Since `ℱ'` is quasi-coherent, there exists `n > 0` such that
`f^n s'` extends to a section `s ∈ Γ(X, ℱ')`; then `f^{m+n} t' − f^n t''' = u(f^n s')`, so `f^{m+n} t'` is the
restriction to `D(f)` of the section `f^n τ + u(f^n s) ∈ Γ(X, ℱ)`, completing the proof.
