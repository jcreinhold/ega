# Translation Glossary

Translation choices for the English EGA I. The first occurrence of a flagged term in each section file is rendered with
a bracketed modern gloss (e.g., `prescheme [modern: scheme]`); subsequent occurrences in the same file use the
historical term alone.

## Quantifier and modality discipline

- $il est imm\acute{e}diat que$ is rendered as `it is immediate that`, never strengthened to `it is obvious that`.
- $on voit aussit\hat{o}t que$ is rendered as `one sees at once that`.
- `il est facile de voir que` is rendered as `it is easy to see that`, preserving the rhetorical force.
- `clairement` / `évidemment` is rendered as `clearly` / `evidently` — never escalated to `obviously` unless EGA uses
  the stronger word.
- `il semble que` is rendered as `it seems that` — never as a stronger claim.

## Punctuation and notation

- Decimal points are used in numerical references (`(2.1.4)`, never `(2,1,4)`).
- French guillemets `« … »` are rendered as English double quotes $"\cdots"$.
- Italics, in EGA reserved for emphasis and for first-occurrence-defines-a-term, are rendered as Markdown italics
  $*\cdots*$ for emphasis and as $**\cdots**$ (bold) for definitional first occurrences inside an unbracketed paragraph.
  (This follows the SGA-I convention.)
- Footnotes in the French source are converted to Markdown footnotes keyed by section and item: $[{}^{N}-M-K]$.

## Notation kept exactly

EGA's symbolic choices that should never be silently modernized:

| EGA                                                            | Modern (informational)  | Comment                                                                                           |
| -------------------------------------------------------------- | ----------------------- | ------------------------------------------------------------------------------------------------- |
| $r(\mathfrak{a})$                                              | $\sqrt{\mathfrak{a}}$   | Radical of an ideal.                                                                              |
| $\mathfrak{J}$, $\mathfrak{p}$, $\mathfrak{q}$, $\mathfrak{m}$ | as such                 | Fraktur letters for ideals; preserve glyphs.                                                      |
| $i^{s}_{M}$                                                    | as such                 | Canonical map $M \to S^{-1}M$; preserve subscript-superscript form.                               |
| $S^{-1}M$                                                      | $S^{-1}M$ (modern same) | Module of fractions.                                                                              |
| $\operatorname{Spec}(A)$                                       | same                    | Prime spectrum.                                                                                   |
| $Spf(A)$                                                       | same                    | Formal spectrum.                                                                                  |
| $Supp(M)$                                                      | same                    | Support of a module.                                                                              |
| $M_{\phi}$                                                     | $\phi_{*} M$            | EGA's notation for restriction of scalars; preserve.                                              |
| $\varinjlim$                                                   | `colim` or $\varinjlim$ | Direct/inductive limit; EGA writes `lim` with a rightward arrow above. We render as $\varinjlim$. |
| $\varprojlim$                                                  | `lim` or $\varprojlim$  | Inverse/projective limit; rendered as $\varprojlim$.                                              |
| $M \otimes_{A} N$                                              | same                    | Tensor product.                                                                                   |
| $\operatorname{Hom}_{A}(M, N)$                                 | same                    | Module of homomorphisms.                                                                          |
| $\Omega^{1}_{X/Y}$                                             | same                    | Sheaf of relative differentials.                                                                  |

## Notes on Grothendieck's voice

EGA sentences are long because they are doing mathematical work: they accumulate hypotheses, comparisons, and viewpoint
shifts in a single architectural unit. In translation we keep long sentences long when their structure carries the
argument. We split a sentence only when English grammar or reader memory would otherwise hide the dependency structure
of the proof. We do not flatten dense paragraphs into friendly textbook prose, and we do not modernize EGA's deliberate
technical vocabulary (`prescheme`, `radical`, `quasi-compact`, `rare`, `dévissage`) without a flagged gloss.
